# Plot temporal partially directed acyclic graph (TPDAG)

Plot temporal partially directed acyclic graph (TPDAG)

## Usage

``` r
# S3 method for class 'tpdag'
plot(x, ...)
```

## Arguments

- x:

  tpdag (temporal partially directed acyclic graph) object to be plotted
  (as outputted from
  [`tpc`](https://bjarkehautop.github.io/causalDisco/reference/tpc.md)).

- ...:

  Further plotting arguments passed along to
  [`plot_tempo_mech`](https://bjarkehautop.github.io/causalDisco/reference/plot_tempo_mech.md).

## Value

No return value, the function is called for its side-effects (plotting).
